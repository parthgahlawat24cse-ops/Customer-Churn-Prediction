# Customer Churn Prediction using Machine Learning

## Project Overview
This project predicts customer churn using Machine Learning techniques on the IBM Telco Customer Churn dataset. The objective is to identify customers who are likely to leave the telecom service and segment them based on their churn probability.

## Dataset
- IBM Telco Customer Churn Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Feature Encoding
- Train-Test Split
- Random Forest Classifier
- Hyperparameter Tuning
- Feature Importance Analysis
- Customer Churn Prediction
- Customer Segmentation using K-Means

## Machine Learning Algorithms
- Random Forest Classifier
- K-Means Clustering
